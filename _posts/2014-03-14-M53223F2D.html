先感谢wenfushan分享的帖子：http://tieba.baidu.com/p/28394
个人感觉这里介绍的方法比较复杂，而且导入ultrapowersaver.pow的时候覆盖了系统自带的节能模式计划。
（有时候超级省电模式的性能显得太低了，这个时候用系统自带的省电模式还是比较均衡的。）
分享下我用的方案：
1. 从http://pan.baidu.com/s/1jG9fzW下载ultrapowersaver.
pow，放到C:\
2. 从菜单打开管理员状态的cmd

3. 执行powercfg /IMPORT C:\ultrapowersaver.pow 
记录下自动生成的一串代码（可以长按命令提示符窗口标题栏，有选择、复制的选项）
4. 新建一个批处理文件“选择电源模式.cmd”
代码如下：
@echo off
echo 1. 切换成高性能模式
echo 2. 切换成节能模式
echo 3. 切换成超级节能模式(需导入ultrapowersaver.pow)
set /p mode="选择1/2/3:"
if %mode%==1 goto part1
if %mode%==2 goto part2
if %mode%==3 goto part3
goto end
:part1
powercfg -setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
goto end
:part2
powercfg -setactive a18 100 6114 0 6114 0 0 435k 0 --:--:-- --:--:-- --:--:-- 497k 41308-3541-4fab-bc81-f71556f20b4a
goto end
:part3
powercfg -setactive 【这里改成刚刚记录下来的那串代码】
:end
（注，也可以建立三个批处理每个执行一种操作，我是嫌桌面上放三个批处理太占地方了）
5. 以后就可以用批处理切换了，输入相应选项就行
